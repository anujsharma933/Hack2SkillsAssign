# 🚀 Hack2Skills Assignment Project

## 📌 Project Overview
This project is built as part of the Hack2Skills assignment. It demonstrates frontend development skills including UI design, responsiveness, and clean coding practices.

## 🛠 Tech Stack
- HTML5
- CSS3 / Tailwind CSS
- JavaScript (Vanilla)
- Git & GitHub

## ⚙️ Setup Instructions

1. Clone the repository:
bash git clone https://github.com/anujsharma933/Hack2SkillsAssign.git 

2. Open the project folder:
bash cd Hack2SkillsAssign 

3. Run the project:
- Open index.html directly  
OR  
- Use Live Server in VS Code
